# 🍅 Pomodoro Timer

A simple and responsive Pomodoro Timer built using HTML, CSS, and JavaScript.

This project is based on the Pomodoro productivity technique where you work for 25 minutes and then take a short 5 minute break. The app also keeps track of completed Pomodoro sessions using Local Storage.

---

## 🚀 Features

- 25 Minute Work Timer
- 5 Minute Break Timer
- Pause & Resume Support
- Reset Timer
- Automatically switches between Work and Break
- Stores completed Pomodoro count in Local Storage
- Responsive UI for mobile devices
- Clean and simple design

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
